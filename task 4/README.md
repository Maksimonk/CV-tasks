# Ответы на вопросы

**Почему RGB-значения задаются в диапазонах от 0 до 255 или от 0 до 1?**  
Диапазон 0–255 соответствует 8‑битному целочисленному представлению цвета, широко используемому в цифровых форматах. Диапазон 0–1 — это нормализованное представление в виде чисел с плавающей точкой, удобное для математических операций, интерполяции и работы с библиотеками вроде Matplotlib.

**Как задать цвет точек на основе скалярной величины?**  
Сначала нормализуют скалярные значения (например, высоту или интенсивность), затем применяют цветовую карту (colormap), такую как `plt.cm.viridis`, чтобы преобразовать их в RGB-цвета.

**В чём разница между статичной (Matplotlib) и интерактивной (Plotly) визуализацией?**  
Matplotlib создаёт статичные изображения, подходящие для публикаций и отчётов. Plotly генерирует интерактивные графики, которые можно вращать, масштабировать и исследовать с помощью всплывающих подсказок.

**Можно ли в Matplotlib использовать другие цветовые карты? Как это сделать?**  
Да. Достаточно указать нужную карту через параметр `cmap`, например: `plt.scatter(..., c=values, cmap='plasma')`.

**Что случится, если не нормализовать значения перед созданием цветов?**  
Без нормализации цвета могут отобразиться некорректно — например, с потерей градаций или выходом за допустимый диапазон, что приведёт к искажённой или пустой визуализации.

**Что такое облако точек и какую информацию оно несёт?**  
Облако точек — это набор точек в 2D или 3D пространстве, содержащий координаты (X, Y, Z) и, при наличии, дополнительные атрибуты: цвет, интенсивность отражённого сигнала, классификацию и др.

**Как устроена цветовая модель RGB?**  
Цвет в модели RGB определяется тремя компонентами — красным (R), зелёным (G) и синим (B). Их комбинация даёт конечный цвет.

**В каких диапазонах должны лежать RGB-значения для корректного отображения?**  
Допустимы два стандарта: целые числа от 0 до 255 или вещественные от 0.0 до 1.0. Оба корректны и легко преобразуются друг в друга.

**Что такое нормализация и зачем она нужна при работе с цветами?**  
Нормализация — это приведение значений к общему масштабу (обычно 0–1), что необходимо для правильного применения цветовых карт и избежания ошибок отображения.

**В чём разница между абсолютным и вычисляемым цветом?**  
Абсолютный цвет фиксирован и не зависит от данных. Вычисляемый цвет генерируется на основе атрибутов точки (например, координаты, расстояния, температуры) и служит для визуального кодирования информации.

**Как с помощью NumPy задать цвет на основе координат точек?**  
Можно использовать одну или несколько координат (например, Z для высоты), нормализовать их и преобразовать в цвета через colormap или прямое отображение в RGB.

**Как вычислить расстояние от каждой точки до центра пространства с помощью NumPy?**  
С помощью функции: `dist = np.linalg.norm(points - center, axis=1)`.

**Как создать цветовой градиент по оси X, Y или Z?**  
Возьмите соответствующую координату, нормализуйте её по минимуму и максимуму, затем примените colormap к полученному массиву.

**Как нормализовать массив вещественных чисел для использования с colormap?**  
Используйте формулу: `norm = (vals - vals.min()) / (vals.max() - vals.min())`.

**Как работает параметр `c=` в функции `plt.scatter()`?**  
Он задаёт значения, по которым определяется цвет каждой точки. Эти значения отображаются в цвета с помощью выбранной цветовой карты (`cmap`).

**Что такое colormap в Matplotlib и как его применить к облаку точек?**  
Colormap — это правило сопоставления скалярных значений цветам. Применяется через аргумент `cmap` в функциях визуализации или напрямую через `plt.cm`.

**Какие типы цветовых карт существуют и как выбрать подходящую?**  
Существуют последовательные (для градиентов), расходящиеся (diverging — для отклонений от среднего) и категориальные (qualitative — для классов). Выбор зависит от природы данных.

**Как отобразить цветовую шкалу рядом с графиком?**  
Используйте `plt.colorbar()`, который автоматически связывается с параметром `c` в функции отображения.

**Как построить 3D-график с помощью `plotly.graph_objects`?**  
Создайте объект `go.Scatter3d`, задав координаты `x`, `y`, `z` и параметры маркера, включая цвет (`color`) и, при необходимости, `colorscale`.

**Как в Plotly вручную задать цвет в формате `'rgb(R,G,B)'`?**  
Передайте список строк вида `'rgb(100, 150, 200)'` в поле `marker.color` при создании `go.Scatter3d`.

**Какие преимущества даёт интерактивная визуализация?**  
Она позволяет в реальном времени исследовать данные: вращать модель, масштабировать область интереса, просматривать точные значения через всплывающие подсказки.

**Какие параметры маркеров можно настроить в Plotly?**  
Размер (`size`), форма (`symbol`), прозрачность (`opacity`), обводка (`line`), цветовая шкала (`colorscale`), направление градиента (`reversescale`) и др.

**Как сохранить облако точек с цветами в файл (например, `.ply` или `.csv`)?**  
Экспортируйте координаты и RGB-компоненты в нужный формат, используя библиотеки вроде `pandas` (для CSV) или `plyfile` (для PLY), убедившись, что цвета сохранены в правильном диапазоне.

**Как использовать цвет для отображения дополнительных данных — например, температуры или классов?**  
Сопоставьте каждому значению (температуре, интенсивности, метке класса) определённый цвет — через colormap для непрерывных данных или фиксированную палитру для категорий.

**Что произойдёт при использовании некорректных RGB-значений (например, >1.0 или <0)?**  
Визуализация может дать сбой или отобразить искажённые цвета. В большинстве библиотек требуется, чтобы значения лежали в допустимом диапазоне, поэтому применяют нормализацию или обрезку (clipping).

**Как применять цветовую визуализацию в практических задачах — анализ ландшафта, сканирование зданий и т.д.?**  
Цветом можно кодировать высоту, отражательную способность, тип поверхности, температуру или другие физические параметры, делая скрытые закономерности видимыми.

**Как визуализировать разные классы точек («земля», «деревья», «здания»)?**  
Назначьте каждому классу уникальный цвет и отобразите точки с этими цветами. При необходимости добавьте легенду для пояснения.

**Что предпочтительнее для презентации: Matplotlib или Plotly? Почему?**  
Для печатных отчётов и публикаций лучше подходит Matplotlib благодаря точному контролю над стилем. Для интерактивных демонстраций и анализа — Plotly, благодаря возможностям взаимодействия с данными.
